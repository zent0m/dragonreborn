# DragonReborn (Handwired Redragon K630)

This is my fork of the QMK repository with my hand-wired K630 added inside. I've used an STM32F401CCU6, an 1306 Display, and a rotary encoder along with an RGB LED for status functionality.

## Documentation

* [See the official documentation on docs.qmk.fm](https://docs.qmk.fm)

The docs are powered by [Docsify](https://docsify.js.org/) and hosted on [GitHub](/docs/). They are also viewable offline; see [Previewing the Documentation](https://docs.qmk.fm/#/contributing?id=previewing-the-documentation) for more details.

You can request changes by making a fork and opening a [pull request](https://github.com/qmk/qmk_firmware/pulls), or by clicking the "Edit this page" link at the bottom of any page.
